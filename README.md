## Getting Started

### Prerequisites

- Node.js `>= 18.x`
- npm `>= 9.x`

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/rishu-portfolio.git
cd rishu-portfolio

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Available Scripts

```bash
npm run dev        # Start development server with HMR
npm run build      # Build for production
npm run preview    # Preview the production build locally
npm run lint       # Run ESLint
```

## Design Philosophy

This portfolio embraces a **neo-brutalist aesthetic** — a design language that rejects the polished, corporate look of typical portfolios in favour of something raw, intentional, and unmistakably human.

- **Paper textures** and rough borders ground the digital in the physical
- **Monochromatic high-contrast palette** forces focus on content and interaction
- **Terminal interface** speaks directly to a technical audience without translation
- **No gradients. No stock photos. No fluff.**
